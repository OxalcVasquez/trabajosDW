AniJS

AniJS nos permite crear estilos animados para nuestros sitio web,Podemos especificar todas las animaciones desde HTML usando una simple sintaxis If-On-Do-To.

Sintaxis de AniJS

If especifica el evento que activará la animación o la manipulación de la clase.
La parte On especifica el elemento cuyos eventos AniJS debe escuchar. Esto puede ser diferente del elemento en el que has configurado el atributo data-anijs.
La parte Do especifica la acción a realizar. Aquí, puedes especificar el nombre de la animación que quieres aplicar, etc.
Por último, la parte To se utiliza para especificar el elemento que necesita ser animado o manipulado.

También podemos utilizar los ganchos Before y After para especificar lo que debería ocurrir antes y después de que AniJS haga lo mencionado dentro de la parte Do.

AniJS nos permite ejecutar una animación activándola en cualquier evento aplicable incluido en la documentación siguiente MDN.

Manipulando clases y elementos de HTML

AniJS te permite hacer más que simplemente animar diferentes elementos. Por ejemplo, puede utilizarlo para agregar, eliminar o alternar clases aplicadas en diferentes elementos. Asimismo, también puedes eliminar elementos HTML o clonarlos sin añadir una sola línea de JavaScript. La biblioteca también te permite recorrer el DOM usando palabras clave reservadas especiales.

Manipulacion de clases

AniJS tiene tres palabras clave reservadas para manipular clases. Estas son $addClass, $removeClass y $toggleClass. De la misma manera, puedes usar palabras clave reservadas como $parent, $ancestors, $closest, $find y $children para recorrer el DOM. Podemos utilizar estos dos conjuntos de palabras clave reservadas para hacer algo, como añadir una determinada clase a todos los elementos children de un elemento, después de que un visitante haga doble clic en ese elemento en particular.
